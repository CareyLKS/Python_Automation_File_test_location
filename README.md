A mini Python automation project:  
When the user download a file, and add a prefix like: lect1.pdf -> Course1_lect1.pdf  
This tool will auto file it to folder Course1, and named in lect1.pdf  
no need to select or move the file by ourselves.  

Version 1:
change the path and alias in config.js for your own setting
use ```python main.py``` to run
